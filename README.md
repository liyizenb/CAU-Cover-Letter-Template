## Introduction / 简介

This is a clean and professional LaTeX cover letter template specifically designed for students and faculty at **China Agricultural University (CAU)**. It is suitable for journal submissions, job applications, or academic inquiries.

这是一个为**中国农业大学 (CAU)** 学子和教职工设计的 LaTeX 投稿信（Cover Letter）模板。风格简洁、专业，适用于学术期刊投稿、求职或学术交流。

---

## Features / 特性

* **Official Branding**: Integrated with CAU logo and standard letterhead layout.
* **Professional Layout**: Built on the `charter` font and `article` class for a classic academic look.
* **Customizable**: Easy-to-edit `.cls` file for margin and style adjustments.
* **Bilingual Support**: Clear structure that works perfectly for international submissions.

* **官方标识**：集成 CAU 校徽及标准页眉横线设计。
* **专业布局**：基于 `charter` 字体和 `article` 类，呈现经典学术感。
* **高度可定制**：通过修改 `.cls` 文件即可轻松调整页边距和样式。
* **结构清晰**：专为国际期刊投稿优化的内容框架。

---

## File Structure / 文件结构

* `template.tex`: The main file where you write your letter content. (主文件，在此填写正文)
* `CoverLetter.cls`: The class file defining the layout and styles. (样式类文件，定义布局)
* `pic/CAU.jpg`: The school logo image. (校徽图片存放位置)

---

## Usage / 使用方法

1.  **Preparation**: Ensure you have a LaTeX environment installed (e.g., TeX Live, MiKTeX) or use [Overleaf](https://www.overleaf.com).
2.  **Logo**: Place the CAU logo image in a folder named `pic/`.
3.  **Edit**: Open `template.tex`, fill in your personal information, the editor's details, and your letter content.
4.  **Compile**: Compile using **pdfLaTeX** or **XeLaTeX**.

1.  **准备**：确保本地已安装 LaTeX 环境（如 TeX Live, MiKTeX）或使用 Overleaf 在线编辑器。
2.  **图片**：将校徽图片放入 `pic/` 文件夹。
3.  **编辑**：打开 `template.tex`，修改个人信息、收件人信息及正文。
4.  **编译**：使用 **pdfLaTeX** 或 **XeLaTeX** 进行编译。
